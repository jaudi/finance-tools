# Finance Tools

A free, open-source finance toolkit built with Streamlit by [EasyVisuals](https://github.com/jaudi/easyvisuals).

No login required. Run it locally or deploy it in one click.

## Tools

### Business & FP&A
| Tool | Description |
|---|---|
| 📈 5-Year Financial Model | Revenue projections, income statement, cash flow, balance sheet and PDF export |
| 💰 Annual Budget | Monthly P&L with sales forecast, direct costs and overheads. Excel template download/upload |

### Markets & Investing
| Tool | Description |
|---|---|
| 📦 Portfolio Analysis | Global index portfolio — expected return, Sharpe ratio, VaR, risk/return scatter |
| 📉 Stock Comparison | Compare two stocks — normalised price, cumulative returns, correlation, key ratios |
| 🛢️ Commodities | Live prices for gold, silver, crude oil, natural gas, copper and wheat |
| 📊 Stock Analysis | Analyse any ticker — price, moving averages, cumulative return, volatility, ratios |

*More tools coming soon.*

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
