# Finance Tools

A collection of free, open-source financial tools built with Streamlit.

## Tools

### 5-Year Financial Model
Set your revenue assumptions, growth rate, COGS, and tax rate. Instantly generates a full income statement, cash flow statement, balance sheet and key ratios — with PDF export.

**Features:**
- Income statement (Revenue, Gross Profit, EBIT, Net Income)
- Cash flow statement (Operating, Investing, Net)
- Balance sheet (Assets, Liabilities, Equity)
- Key financial ratios (Gross Margin, Operating Margin, Net Margin, ROA, ROE)
- Interactive charts powered by Plotly
- PDF export with branded report

*More tools coming soon.*

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
